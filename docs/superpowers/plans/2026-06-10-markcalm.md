# MarkCalm Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a native macOS read-only markdown viewer that opens `.md` files one window per file, with calm UI, optional reading progress, and system light/dark support.

**Architecture:** SwiftUI document-based app (`DocumentGroup`) with MarkdownUI for GFM rendering, Yams for frontmatter stripping, and `@AppStorage` for settings. No network calls.

**Tech Stack:** Swift, SwiftUI, MarkdownUI, Yams, macOS 14+

**Spec:** [`docs/superpowers/specs/2026-06-10-markcalm-design.md`](../specs/2026-06-10-markcalm-design.md)

- [ ] **Step 3: Implement**

```swift
import Foundation
import Yams

struct ProcessedMarkdown {
    let body: String
    let hadFrontmatter: Bool
}

enum MarkdownPipeline {
    static func process(_ raw: String) -> ProcessedMarkdown {
        guard raw.hasPrefix("---") else {
            return ProcessedMarkdown(body: raw, hadFrontmatter: false)
        }
        let lines = raw.split(separator: "\n", omittingEmptySubsequences: false)
        guard lines.count >= 2 else {
            return ProcessedMarkdown(body: raw, hadFrontmatter: false)
        }
        var endIndex: Int?
        for i in 1..<lines.count {
            if lines[i] == "---" {
                endIndex = i
                break
            }
        }
        guard let endIndex else {
            return ProcessedMarkdown(body: raw, hadFrontmatter: false)
        }
        let yaml = lines[1..<endIndex].joined(separator: "\n")
        do {
            _ = try Yams.load(yaml: yaml)
        } catch {
            return ProcessedMarkdown(body: raw, hadFrontmatter: false)
        }
        let bodyStart = lines.index(after: endIndex)
        let body = lines[bodyStart...].joined(separator: "\n")
        return ProcessedMarkdown(body: body, hadFrontmatter: true)
    }
}
```

- [ ] **Step 4:** Run tests — expect PASS.

### Task 3: Document model

**Files:**
- Create: `MarkCalm/MarkdownDocument.swift`

- [ ] **Step 1:** Implement read-only `FileDocument`:

```swift
import SwiftUI
import UniformTypeIdentifiers

struct MarkdownDocument: FileDocument {
    static var readableContentTypes: [UTType] {
        [UTType(filenameExtension: "md")!, .plainText]
    }

    var rawContent: String
    var fileURL: URL?

    init(text: String = "") {
        rawContent = text
    }

    init(configuration: ReadConfiguration) throws {
        guard let data = configuration.file.regularFileContents,
              let text = String(data: data, encoding: .utf8) else {
            throw CocoaError(.fileReadCorruptFile)
        }
        rawContent = text
        fileURL = configuration.file.fileURL
    }

    func fileWrapper(configuration: WriteConfiguration) throws -> FileWrapper {
        FileWrapper(regularFileWithContents: Data(rawContent.utf8))
    }

    var processed: ProcessedMarkdown {
        MarkdownPipeline.process(rawContent)
    }

    var displayName: String {
        fileURL?.deletingPathExtension().lastPathComponent ?? "Untitled"
    }
}
```

Note: document is technically writable but UI provides no edit affordances (read-only in practice).
